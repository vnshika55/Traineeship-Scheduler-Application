#📅 Traineeship Scheduler Application

## 🚀 Overview
The Traineeship Scheduler is a web-based application designed to automate the process of creating training schedules for different qualifications. It simplifies planning by considering units, sequencing, and public holidays.

## 🎯 Features
- 📘 Select qualifications dynamically
- 🧩 Automatic unit scheduling based on sequence
- 📅 Considers public holidays while generating schedules
- ⚡ Fast and efficient schedule generation
- 🌐 Deployed using Streamlit Cloud for easy access

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas

## 📂 Project Structure
- `app.py` – Main Streamlit application
- `scheduler.py` – Core scheduling logic
- `data/` – Contains CSV datasets (units, qualifications, holidays)

This application is useful for training organizations (RTOs) to efficiently manage and generate structured training plans.
